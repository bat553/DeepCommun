Foire Aux Questions

Q. Le Deepcommun est-il illégal ?

R. Non, du moins d'un point de vue de la loi non. Pour l'installer vous devez avoir l'autorisation

du directeur de votre établissement.

Q. Y a-t-il des scripts de piratages ?

R. Non, aucun le Deepcommun est une moyenne facilité de créer un échange lieux d'échanges entre les élèves. Il n'y a donc pas besoin d'utiliser des techniques illégales.

Q. Le Deepcommun a-t-il été testé ?

R.Oui, lors du développement nous avons créé un environnement virtuel étant basée sur les infrastructures scolaires type. ( Windows Serveur 2008 R2, Windows XP/7 ).

Q. Un source code ?

R. Non pas pour le moment.
